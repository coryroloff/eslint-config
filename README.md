# @coryroloff/eslint-config

ESLint configured for me!

## Installation

```bash
npm install @coryroloff/eslint-config@coryroloff/eslint-config
```

## Usage

```json
{
  "scripts": {
    "lint": "eslint --ignore-pattern !.eslintrc.js .eslintrc.js src",
    "lint:fix": "npm run lint -- --fix"
  }
}
```
